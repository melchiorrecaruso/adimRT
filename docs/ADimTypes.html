Unit ADimTypes

Description
Classes, Interfaces, Objects and Records

Description

ADim record types and data structures.

Defines the core record types and data structures used throughout the ADimPas library, including:

  • TDimension — the SI dimension exponent record, storing the integer exponents of the eight SI base units and the steradian.
