Public function Cross(const AVector: TR3Vector): TR3Vector;

Returns the cross product of two 3-component real vectors. u×v = (u₂v₃ - u₃v₂, u₃v₁ - u₁v₃, u₁v₂ - u₂v₁) The result is a vector perpendicular to both operands.
