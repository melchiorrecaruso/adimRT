type TUnitHelper = record helper for TUnit

Description

Record helper for TUnit providing conversion and formatting operations for all supported quantity types.

This helper centralises the logic for:

  • Extracting the raw numerical value of a quantity expressed in a given unit and prefix, via the ToFloat, ToComplex, ToVector and ToMatrix family of methods.

  • Formatting a quantity as a compact string with unit symbol, via the ToString family of methods.

  • Formatting a quantity as a verbose string with full unit name, via the ToVerboseString family of methods.

  • Computing the scaled numerical value of a raw dimensionless type for a given prefix, via the GetValue family of methods.

All methods that accept APrefixes apply the corresponding SI prefix scaling factor to the numerical value before returning or formatting.
